# Glossary (shared vocabulary)

- **Public API**: externally visible surface used by other modules (headers / RPC / schemas / asset formats)
- **Contract**: semantic requirements (threading, lifetime, ownership, error behavior, performance)
- **Owner**: maintainer of a module contract; approves breaking changes
- **Breaking Change**: downstream users must modify code/data to remain compatible
- **Stub-first**: ship compile/linkable stubs early so others can integrate in parallel
- **RHI**: Render Hardware Interface
